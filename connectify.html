Event name : Connectify-2048

Event type : On-stage Event

Event date : 30th January

Event Time : 8.15 pm

Event venue : Lords Arena

Event description : Love solving puzzles?

Come and play the game of connecting the several pieces and forming the big picture which leads you to the answer!!

Connectify-2048 is the game where you need no former preparations, only need to have the knack of joining the image clues and give the answer.

RULES:
> You will be given several image clues. You have to connect them together and think of the answer which the image clues together represent.
> Teams of 3 or less can participate.
> Top few teams will be qualifying for the next round.
